**_2.What is pygame_**:
  * pygame提供display,sound,image,text,event幫助製作遊戲
  * pygame可做出2-d小遊戲
  * pygame偵測使用者用keyboard,joystick,mouse控制遊戲
  * pygame提供許多內建的game object來製遊戲<br><br>
